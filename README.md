
# 中文名转拼音字典

```
usage: cn2pydict.py [-h] (-n NAME | -N NAMELIST) [-o OUTFILE]

中文名转用户名字典

options:
  -h, --help   show this help message and exit
  -n NAME      姓名，可以用逗号分隔。
  -N NAMELIST  姓名列表文件
  -o OUTFILE   输出生成的用户名字典
```

```
python cn2pydict.py -n 张三丰,李四,王麻子,卡拉肖克

python cn2pydict.py -N topnames_20.txt -o usernames_20.txt
```
